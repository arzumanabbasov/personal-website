# personal-website

Minimal personal website for Arzuman Abbasov, Data Scientist.

This project follows a clean, simple philosophy — no heavy frameworks, no unnecessary dependencies, just HTML and a bit of CSS to present essential information clearly and accessibly.

The website includes:

* **Home** – a professional introduction and overview
* **Projects** – individual project case studies
* A **Resume** link in the navbar

The site is open-source, lightweight, and easy to read — both for humans and machines.

---

## Structure

* `index.html` – Home page
* `projects.html` – Projects list
* `projects/*.html` – Individual project pages
* `styles.css` – Simple styling

---
## Philosophy

This website is designed for HR and recruiters. It prioritizes:

* clarity
* fast load times
* minimal design
* easy maintainability

No JavaScript, no build tools, no frameworks — just semantic HTML and straightforward CSS.
